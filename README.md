# Justworks landing page

A bespoke, single-page read prepared for the Justworks team at eMerge (booth 1125). The page uses Justworks' current visual system (the 2023 Order-designed identity: deep navy #15246D, Dodger Blue #1E98FF, flame orange #DF5A31 as underline accent, Buttercup #F3B715 as a warm highlight), renders the wordmark at the top, and echoes the Oately-era typography with Inter as the web-safe stand-in that Justworks' own brand notes recommend. The headline, opening, frame, case, methodology, and CTA are carried verbatim from the fit analysis Section 7 pitch body. The sole Eduba mark lives in the footer. The CTA routes to Matt Creamer's 30-minute Calendly. No engagement pricing appears on the page.
